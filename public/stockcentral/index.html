Help & Keyboard Shortcuts

Quick reference for using the inventory dashboard.

How to Use
  • Use the filter pills at the top to filter by All, In Stock, Out of Stock, Unmanaged, or by location (Boca, Jupiter, Dropship)
  • Use the search bar to find products by name, SKU, or supplier
  • Use the toolbar filters to narrow by stock status, source, or supplier
  • Click column headers to sort the table
  • Rows are color-coded: ■ green = in stock, ■ amber = low, ■ red = out of stock
  • Source column shows Managed vs Unmanaged stock items
  • Boca / Jupiter / Dropship columns show per-location stock quantities
  • Use the Location filter to view only products at a specific location
  • Data auto-refreshes every 4 hours. Press R to refresh manually.

Contact your ops team for API configuration or connection issues.
